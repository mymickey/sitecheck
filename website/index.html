macOS menubar utility

SiteCheck

SiteCheck is a focused macOS utility for monitoring route quality from your current network path. It helps you check site connectivity, inspect DNS exit results, and view your current public IP directly from the menubar.

Connectivity checks for your selected targets

DNS test results with ISP and exit country visibility

Fast My IP lookup from the tray menu
